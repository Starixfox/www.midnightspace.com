FULL CATALOGUE

All modules,
in order.

Numbered roughly in build order. Each module compounds the value of the one before it — start with data, end with intelligence.

01

Funding & Program Database

GrantsAcceleratorsTendersPublic programs
A manually curated database of space-related funding calls, grants, accelerators, and public programs. Focused on Saudi Arabia and the GCC, with selected global entries open to regional actors.
Prototype live
02

Basic Dashboards

KPIsChartsSummary views
Charts and summary views built from the program database — designed for leadership-level overviews rather than deep data exploration. The Saudi Space Opportunity Hub runs on this.
Prototype live
03

Global Space Funding View

EuropeNorth AmericaAsiaEmerging markets
Extending coverage beyond the GCC to include major space-funding programs from Europe, North America, Asia, and emerging markets — with the same curation discipline.
Planned
04

API Access

Structured dataInternal toolsWorkflows
Structured data access for organisations that want to integrate Midnight Space data into their own internal tools, reports, or workflows. Versioned, rate-limited, documented.
Planned
05

Custom Views for Agencies

Sector filtersRegional cutsMandate-aligned
Tailored dashboard configurations for specific agency mandates — filtered to the sectors, regions, and program types each team actually cares about.
Planned
06

Signals & Alerts

New programsDeadlinesStatus changes
Automated notifications when new programs open, deadlines approach, or data changes — so the right people hear about the right opportunities in time to act.
Planned
07

Exports & Reports

CSVPDF briefingsCurated views
Export filtered datasets, generate summary reports, and share curated views with stakeholders who don't need a login. Built for how briefings actually circulate.
Planned

Ecosystem Intelligence

TrendsScenario viewsGap analysis
Deeper analytics — trend views over time, scenario comparisons across regions, and gap analysis between programs. The layer that turns data into a point of view.
